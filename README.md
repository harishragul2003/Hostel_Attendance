# Hostel Attendance Management System

A modern web-based system for managing hostel student attendance and meal tracking.

## Features

- **User Authentication**
  - Student and Admin roles
  - Secure login system

- **Student Dashboard**
  - Mark attendance for meals
  - View attendance history
  - Modern UI with animations
  - Real-time status updates

- **Admin Features**
  - Manage student records
  - View attendance reports
  - Generate billing reports
  - Meal schedule management

- **Reports**
  - Monthly attendance reports
  - Detailed student reports
  - Billing summaries
  - Attendance analytics

## Tech Stack

- Python/Flask
- SQLAlchemy
- HTML/CSS
- Bootstrap 5
- Font Awesome
- JavaScript

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/hostel-attendance.git
cd hostel-attendance
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```
